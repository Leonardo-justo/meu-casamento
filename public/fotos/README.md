Adicione aqui as fotos do casal.

Como usar:
- Copie os arquivos de imagem para esta pasta, por exemplo: `noivado-01.jpg`.
- No painel admin em `Galeria`, use a URL no formato: `/fotos/noivado-01.jpg`.
- Você pode organizar a ordem de exibição pelo campo `order` no admin.

Formatos recomendados:
- `.jpg`, `.jpeg`, `.png`, `.webp`

Dica:
- Prefira arquivos otimizados para web para o site carregar mais rápido.
